# 👔 dpClothing UI

Modern and elegant clothing menu for FiveM servers with a sleek dark theme.

---

## ✨ Features

- 🎨 Modern dark UI with smooth animations
- 👕 Multiple categories (Head, Body, Accessories)
- 🎯 Easy to use interface
- 🔊 Sound effects & camera system
- ⌨️ Keyboard support (ESC to close)
- 🌐 Multi-language support

---

## 📦 Installation

1. Download the resource
2. Place `dpclothingui` in your `resources` folder
3. Add `ensure dpclothingui` to `server.cfg`
4. Restart server

---

## 🎮 Usage

- **Open Menu**: Press `Y` (configurable)
- **Close Menu**: Press `ESC` or click close button
- **Select Item**: Click any clothing icon

---

## ⚙️ Configuration

Edit `Client/Config.lua`:

```lua
Config = {
    Language = "en",
    NUI = {
        DefultKey = 'Y',    -- Menu key
        Sound = true,       -- Sound effects
        Camera = true,      -- Camera system
    }
}
```

---

## 🎨 Clothing Categories

**HEAD**: Hat, Hair, Glasses, Mask, Earrings  
**BODY**: Jacket, Shirt, Vest, Pants, Shoes  
**ACCESSORIES**: Necklace, Watch, Gloves, Bag
